Jail Tracker

codyssey.kr SECOM에 출입하는 모든 길드 멤버들의 출입 기록을 자동 수집 & 시각화하는 통합 랭킹 대시보드입니다. 로그인이 필요한 실시간 대시보드는 Codespaces에서 1클릭으로 띄울 수 있고, 이 GitHub Pages는 소개 페이지 + 최신 랭킹 미리보기 용도입니다.

🔒 공개 페이지 프라이버시 안내

이 GitHub Pages는 익명 통계 뷰어로, 다음 정보만 공개됩니다: 이름, 레벨, 길드점수, 일별/월별 인정 학습시간, 입퇴실 시간 통계.

이메일 주소, 내부 회원번호(ID) 등 개인식별 정보는 배포 전 자동으로 제거됩니다. 자세한 세션/자신의 상세 기록은 Codespaces로 연 라이브 대시보드에서 로그인 후 확인하세요.

⚠️ 이 페이지는 인터넷에 공개됩니다. 외부에 링크 공유를 원치 않으면 GitHub Pages를 비활성화하거나, 리포를 Private으로 유지하고 Codespaces에서만 대시보드를 사용하세요.

🚀 빠른 시작

  1. 상단 🚀 Codespaces로 열기 버튼을 클릭해 Codespace를 생성합니다 (약 2분 소요).
  2. 자동으로 서버가 기동되면 브라우저 미리보기 창이 열립니다.
  3. codyssey.kr 아이디/비밀번호로 로그인하면 대시보드가 뜹니다.
  4. 로그인과 동시에 세션이 자동으로 GitHub Secret에 동기화되어, 매시 GitHub Actions가 자동으로 데이터를 수집합니다.

⚠️ 참고: GitHub Pages는 정적 호스팅만 지원해 백엔드 로그인/API 프록시 기능이 동작하지 않습니다. 라이브 대시보드는 반드시 Codespaces 또는 로컬에서 서버를 띄워 사용해주세요.
