# 🎯 SECOM 대시보드

코디시 길드 멤버들의 출입/학습 기록을 시각화하는 웹 대시보드입니다.

## 기능

- **실시간 멤버 랭킹**: 이번 달 누적 인정시간 순으로 정렬
- **캘린더 히트맵**: 날짜를 누르면 그날 출입한 멤버를 시간순으로 조회
- **멤버 상세**: 이름 클릭 시 일별 세션(입/퇴실 시간) 모두 확인
- **현재 출입중**: 퇴실 기록이 없는 오늘자 세션 보유자 실시간 표시 (1분마다 자동 갱신)
- **통계 차트** (Chart.js):
  - 시간대별 체류 분포 → 피크 타임 확인
  - 요일별 총 학습시간
  - 주차별 출입 현황 → 사람이 가장 많은 주차 확인
- **최다 기록 배지**: 최다 공부 멤버, 최다 출석일, 최다 학습시간일, 피크 타임
- 일일 12시간 상한, 퇴실 미기록 세션 모두 처리

## 실행 방법

```bash
cd dashboard
npm install    # 최초 1회
node server.js
```

브라우저에서 http://localhost:3000 접속

## 상단 컨트롤

- 조회 범위는 **3·4·5·6길드 통합**으로 고정되며 길드 선택 UI는 제공하지 않습니다.
- **연도 / 월**: 과거 월 데이터도 조회 가능
- **불러오기 / 🔄 새로고침**: 수동 갱신 (2분간 자동 캐시)

## 파일 구조

```
dashboard/
├── server.js        # Express 백엔드 (API 프록시 + 집계)
├── package.json
└── public/
    └── index.html   # 프론트엔드 단일 페이지 (Chart.js CDN)
```

## 환경변수 (옵션)

```bash
PORT=3000 node server.js
# 길드 범위는 서버에서 3,4,5,6으로 고정됩니다.
```

## 주의사항

- 서버가 `https://api.usr.codyssey.kr` 에 직접 요청을 보내 데이터를 가져옵니다.
- 외부 API 응답 속도에 따라 최초 로딩에 수십 초가 걸릴 수 있습니다 (멤버 1명당 1회 요청, 5명씩 병렬).
- 5분간 동일 요청 결과를 메모리 캐시하므로 반복 조회가 빠릅니다.
- "현재 출입 중" 판정은 오늘 날짜의 퇴실시간이 비어있는 세션 기준입니다 (KST 기준).
